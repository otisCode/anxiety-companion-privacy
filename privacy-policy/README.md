# Privacy Policy for Anxiety Exposure Tracker

This repository hosts the privacy policy for the Anxiety Exposure Tracker iOS app.

## Setup Instructions

### Quick Setup (5 minutes)

1. **Create a new GitHub repository**
   - Go to https://github.com/new
   - Name it: `anxiety-exposure-privacy` (or any name you prefer)
   - Make it Public
   - Don't initialize with README
   - Click "Create repository"

2. **Upload the index.html file**
   - Click "uploading an existing file"
   - Drag the `index.html` file
   - Commit with message "Add privacy policy"

3. **Enable GitHub Pages**
   - Go to Settings tab
   - Scroll down to "Pages" in the left sidebar
   - Under "Source", select "Deploy from a branch"
   - Under "Branch", select "main" (or master)
   - Select "/ (root)"
   - Click Save

4. **Your privacy policy URL will be:**
   ```
   https://[your-github-username].github.io/anxiety-exposure-privacy/
   ```
   (It takes 2-10 minutes to go live)

### Example URLs

If your GitHub username is `dustinfloer`, your URL would be:
- Repository: `https://github.com/dustinfloer/anxiety-exposure-privacy`
- Privacy Policy: `https://dustinfloer.github.io/anxiety-exposure-privacy/`

### What to Put in App Store Connect

In the Privacy Policy URL field, enter your GitHub Pages URL:
```
https://[your-username].github.io/anxiety-exposure-privacy/
```

### Support URL

You can use the same URL for your Support URL, or create a separate page with FAQs.

### Updating the Policy

To update the privacy policy:
1. Edit the `index.html` file directly on GitHub
2. Update the "Last Updated" date
3. Commit the changes
4. Changes go live automatically in a few minutes
